\section{Plots}
You can construct plots with latex as well using pgfplots package. Read \url{https://www.sharelatex.com/learn/Pgfplots_package} for more details. Below is a sample plot.

\begin{figure}[ht]
  \centering
  \begin{tikzpicture}
    \begin{axis}
      \addplot[color=red]{exp(x)};
    \end{axis}
  \end{tikzpicture}
\end{figure}
